# Python-GUI-chat
## Cosa?
Un applicazione per creare una chatroom su una rete locale tramite socket. 
## Come?
#### Server:
  - Modificare host/port (defaults: localhost/4444)
  - Eseguire [server.py](server.py)
#### Client:
  - Modificare host/port (defaults: localhost/4444)
  - Eseguire [client.py](client.py)
  - Connettersi al server
